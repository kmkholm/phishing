Phishing URL Detection API

API Endpoints

  • /api/check-url - POST endpoint to check if a URL is phishing
  • /api/analyze-url - POST endpoint to analyze URL features
  • /api/health - GET endpoint to check server health status

Example Request

curl -X POST http://localhost:5000/api/check-url \
    -H "Content-Type: application/json" \
    -d '{"url": "https://example.com"}'

Example Response

{
    "url": "https://example.com",
    "is_phishing": false,
    "phishing_probability": 0.01,
    "confidence": 0.99,
    "prediction": "Legitimate",
    "risk_factors": [],
    "risk_level": "Low"
}
